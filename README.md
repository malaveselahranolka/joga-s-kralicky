# Jóga s králíčky 🐰

Web studia klidu, kde po lekcích jógy volně pobíhají domácí králíčci.

Jednostránkový statický web — `index.html` se všemi styly inline, logo v `assets/logo.svg`.

## Spuštění lokálně

Stačí otevřít `index.html` v prohlížeči, nebo posloužit složku jakýmkoliv statickým serverem:

```bash
npx serve .
```

## Nasazení

Publikováno přes GitHub Pages z větve `main` (kořen repozitáře).
